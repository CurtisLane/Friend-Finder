# Friend-Finder
A full stack web application which allows a user to find a friend based on compatibility.

## Deployed on Heroku!
[Friend Finder](https://friend-finder064.herokuapp.com/)

## Technologies Used:
Node.js, Express.js, HTML5, CSS3, and Bootstrap.
